# نمونہ مارک ڈاؤن فائل

## تعارف

یہ مارک ڈاؤن ترجمہ کا ایک مظاہرہ ہے۔ ایکسٹینشن عنوانات، پیراگراف اور دیگر مارک ڈاؤن عناصر کا ترجمہ کر سکتی ہے۔

## خصوصیات

- مارک ڈاؤن میں باقاعدہ متن کا ترجمہ کرتا ہے۔
- مارک ڈاؤن فارمیٹنگ کو محفوظ رکھتا ہے۔
- فہرستوں اور گھریلو عناصر کو ہینڈل کرتا ہے۔

## کوڈ کی مثالیں۔

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```
