# Sample Markdown File

## Introduction

This is an example of Markdown translation. This extension can translate headings, paragraphs, and other Markdown elements.

## features

- Translates normal text into Markdown
- Preserves Markdown formatting
- Handles lists and nested elements

## code example

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> This is a block quote that will be translated.
