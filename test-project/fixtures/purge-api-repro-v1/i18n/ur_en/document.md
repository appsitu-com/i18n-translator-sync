# Sample Markdown file

## Introduction

This is a demonstration of Markdown translation. The extension can translate headings, paragraphs, and other Markdown elements.

## Features

- Translates regular text into Markdown.
- Markdown preserves formatting.
- Handles lists and nested elements.

## Code examples

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> This is a block quote that will be translated.
