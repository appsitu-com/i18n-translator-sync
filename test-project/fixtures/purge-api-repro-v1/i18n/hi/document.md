# नमूना मार्कडाउन फ़ाइल

## परिचय

यह मार्कडाउन अनुवाद का एक उदाहरण है। यह एक्सटेंशन शीर्षकों, पैराग्राफों और अन्य मार्कडाउन तत्वों का अनुवाद कर सकता है।

## विशेषताएँ

- मार्कडाउन में सामान्य टेक्स्ट का अनुवाद करता है
- मार्कडाउन फॉर्मेटिंग को सुरक्षित रखता है
- सूचियों और नेस्टेड तत्वों को संभालता है
