# Beispiel-Markdown-Datei

## Einführung

Dies ist eine Demonstration der Markdown-Übersetzung. Die Erweiterung kann Überschriften, Absätze und andere Markdown-Elemente übersetzen.

## Funktionen

- Übersetzt normalen Text in Markdown
- Behält die Markdown-Formatierung bei
- Verarbeitet Listen und verschachtelte Elemente

## Code-Beispiele

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> Dies ist ein Zitat, das übersetzt wird.
