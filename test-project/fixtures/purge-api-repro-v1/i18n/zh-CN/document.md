# Markdown 示例文件

## 导言

这是 Markdown 转换的演示。该扩展程序可以转换标题、段落和其他 Markdown 元素。

## 功能

- 将Markdown中的普通文本转换为
- 保留 Markdown 格式
- 支持列表和嵌套元素

## 代码示例

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> 这是一段将被翻译的引用内容。
