# Archivo de Markdown de Ejemplo

## Introducción

Esto es una demostración de la traducción de Markdown. La extensión puede traducir encabezados, párrafos y otros elementos de Markdown.

## Características

- Traduce texto normal en Markdown
- Preserva el formato Markdown
- Maneja listas y elementos anidados

## Ejemplos de código

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> Esta es una cita bloqueada que se traducirá.
