# Exemple de fichier Markdown

## Introduction

Il s'agit d'une démonstration de la traduction de Markdown. L'extension peut traduire les titres, les paragraphes et d'autres éléments Markdown.

## Caractéristiques

- Traduit du texte normal en Markdown
- Préserve le formatage Markdown
- Gestion des listes et des éléments imbriqués

## Exemples de codes

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> Il s'agit d'une citation en bloc qui sera traduite.
