# Markdown Sample File

## introductory

This is a demonstration of Markdown conversion. This extension can convert headings, paragraphs, and other Markdown elements.

## Features

- Convert plain text in Markdown to
- Preserve Markdown formatting
- Supports lists and nested elements

## Code example

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> This is a quote that will be translated.
