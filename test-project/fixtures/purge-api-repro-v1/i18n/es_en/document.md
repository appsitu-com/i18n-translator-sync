# Example Markdown File

## Introduction

This is a demonstration of Markdown's translation. The extension can translate headings, paragraphs, and other Markdown elements.

## Features

- Translate regular text in Markdown
- Preserve Markdown formatting
- Handle lists and nested items

## Code Examples

```javascript
// Code blocks are not translated
const message = "Hello, World!";
console.log(message);
```

> This is a blocked appointment that will be translated.
